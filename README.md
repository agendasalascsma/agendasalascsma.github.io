# Solicitudes de Salas - Colegio

Este sitio web muestra un calendario embebido de Google Calendar que permite a docentes y personal visualizar las reservas de salas del colegio en tiempo real.

## Contenido

- `index.html`: Página principal del sitio.

- `index.js`: Lógica adicional (vacío por ahora).
- `README.md`: Este archivo.
